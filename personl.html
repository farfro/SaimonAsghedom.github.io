Projects

Lexical Analyzer & Parser

The project was about implementing a program that takes the source code of a simple language's program as input and breaks it down into lexical tokens. It identifies keywords, identifiers, and operators of the programming language and creates a token for each of them. The parser then takes the tokens generated by the lexical analyzer and analyzes the syntax of the program to create a parse tree. It checks whether the tokens are arranged in a way that conforms to the syntax of the programming language.

Pixtoral

Pixtoral is a real-time collaborative platform for drawing and painting, inspired by Reddit's r/place project. The project was developed by a team of four members using React, Express, Node.js, and PostgreSQL, with the goal of creating a fun and interactive platform where users can collaborate with others to create beautiful art. As part of the development team, I was responsible for creating the server-side API using Express and Node.js. The API handled requests from the client and updated the pixel colors in the PostgreSQL database in real-time. I also worked with the team to plan and manage the project using agile development methodologies and collaborated with the team using version control to ensure efficient and effective teamwork.

Rental Bulletin Board

The Application is a software system designed for landlords or property managers to advertise their available rental properties to potential tenants. The application provides a console-based platform where landlords can provide information such as the location of the rental property, the rental price, and the contact information of the landlord or property manager. Potential tenants can browse the rental bulletin board to find rental properties that meet their requirements and contact the landlord or property manager directly to schedule a viewing or apply for the rental. The project demonstrates advanced use of polymorphism and inheritance, showcasing proficiency in object-oriented programming design. Additionally, the system utilizes a binary search tree (BST) data structure to efficiently store and retrieve data, ensuring optimal performance and scalability. Furthermore, the project showcases my expertise in developing reliable and robust applications using exception handling and proper documentation practices.
